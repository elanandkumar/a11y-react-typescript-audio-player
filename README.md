An accessible audio player created using react with typescript. Following accessibility features are available:
- voice over: When tabbed to the control, voice over can be heard
- keyboard navigation: Use tab to set focus to play/pause or seekbar. Then following features can be used
  - Arrow keys can be used to seek to the audio
    - left/right: To seek by small amount
    - top/down: To seek faster
  - space/enter can be used to play/pause
